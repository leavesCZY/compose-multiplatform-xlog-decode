# compose-multiplatform-xlog-decode

一个跨平台的 xlog 日志解析工具，支持 Windows、MacOS、Linux

包含两个功能点：

- 解压和解密 xlog 日志文件
- 生成加密 xlog 日志文件所需要的公钥和私钥

Windows 系统推荐使用免安装版本 `compose-multiplatform-xlog-decode-windows-x86_64.zip`
